Par exemple ce site que vous consultez actuellement contient une page de code CSS,il traite par exemple de la mise en forme du texte s'il est collé ou non à la bordure de la page web ou encore à créer l'encadré dans lequel vous lisez actuellement ce texte, mais on peut aussi modifier la police de l ecriture ! Pour finir j'ajouterai simplement que le CSS ne sert pas à coder la structure d'un site web il est donc indispensable de le combiner avec un autre langage informatique de structure de texte comme l HTML
